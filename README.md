Para usar este código es necesario instalar Arduino ide de la siguiente página:  https://www.arduino.cc/en/software/
En esta página es recomendable usar la versión 2.3.6 ya que es la última y más completa, aparte de que es más fácil instalar las librerías
Las librerías a instalar son las siguientes:
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

Si queremos instalarlas manualmente los links son los siguientes:
Wire.h: https://docs.arduino.cc/language-reference/en/functions/communication/wire/
LiquidCrystal_I2C: https://docs.arduino.cc/libraries/liquidcrystal-i2c/

Una vez ya teniendo todo, solo queda armar el circuito y subir el código en una tarjeta ARDUINO UNO compatible.
Vease el listado de tarjetas compatibles en: https://docs.arduino.cc
